# The Pixel & Lens Co.

A photography portfolio website for Lois Coutinho, built with HTML and ready to publish for free with GitHub Pages.

## Live Site

No custom domain is needed. After publishing with GitHub Pages, GitHub will give you a free public link like:

`https://your-username.github.io/your-repository-name/`

## About

The Pixel & Lens Co. is a portfolio website for wedding, event, couple, family, and corporate photography by Lois Coutinho.

The site can be viewed locally by opening `index.html` in a browser, or published online for free using GitHub Pages.

## Files

- `index.html` - the main website page
- `README.md` - this project description

## Publish With GitHub Pages

1. Create a new public repository on GitHub.
2. Upload `index.html` and this `README.md` file.
3. Open the repository settings.
4. Go to **Pages**.
5. Under **Build and deployment**, choose:
   - Source: **Deploy from a branch**
   - Branch: **main**
   - Folder: **/root**
6. Save the settings.

GitHub will publish the site and show a public link after a few minutes.

You can use the free GitHub Pages link as your public website address. A custom domain is optional and can be added later.

## Notes

If the page uses images, styles, or scripts, upload those files too. Make sure the links inside `index.html` use relative paths, not paths from your computer like `C:\Users\...`.
